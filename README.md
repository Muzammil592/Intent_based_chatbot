Day 1 — Simple Prototype + Intent RecognitionThe Day 1 phase established a clean, modular, production-ready system architecture completely isolated from high-level orchestration frameworks to showcase deep code-level engineering.1. Statistical Tokenization & Semantic LemmatizationRaw user text inputs undergo string normalization and regex cleaning to eliminate symbols. Text streams are passed through the NLTK tokenization engine to separate words cleanly. Words are then processed through the NLTK WordNetLemmatizer, reducing inflectional forms to their base dictionary forms (e.g., "running" or "runs" map to "run"). This ensures the statistical engine matches structural intents regardless of grammatical tense variations.2. Vector Matrix & Feature ExtractionTo convert text strings into dense mathematical vector parameters that machine learning algorithms can calculate, the processed tokens are mapped onto a TF-IDF (Term Frequency-Inverse Document Frequency) Vector Space via Scikit-Learn. This measures the unique weight and statistical importance of specific keywords across the entire intent sample matrix.3. Convex Optimization via Logistic RegressionThe classification model implements a Logistic Regression algorithm configured with high iteration limits to construct a perfectly convex optimization surface. This mathematical framework guarantees that Gradient Descent reaches the true global minimum when identifying user intents. It outputs exact prediction labels (greet, schedule, weather, reminder, farewell, small_talk) alongside explicit confidence probabilities.4. Anti-Hallucination Fallback & Containerized TTY LoopAn asynchronous interactive runtime engine (chatbot.py) continuously reads user inputs via a CLI interface. It features an anti-hallucination guardrail that intercepts predictions: if maximum confidence scores fall below a strict threshold ($<0.35$), it drops the intent and defaults to a safe fallback response rather than guessing. The final architecture is packaged into a multi-stage production Dockerfile that pre-downloads linguistic corpora and maps your terminal directly into an isolated interactive container.
